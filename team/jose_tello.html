The particle system is able to render +100.000 particles at 60fps in the target hardware using instanced rendering. Thought this was my biggest implication in this system, I also participaded in the design of the system and coded some effects for it.

Ui rendering system




The ui system is based on a canvas that holds all ui elements, which will be rendered to the game camera. The ui elements can be moved using the 2d transform component.

It includes a text system that supports ttf fonts. I also worked on ui easy-ins, fade-ins, and the grenade reload effect.
